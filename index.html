Why Arachnid?

The Dark Net:
It pervades culture in how "scary" it is, but in reality it's just a bunch of programmers making a bunch of bits flip. The idea of Arachnid is to fill in the boots where Dark Nets (such as Tor) fail and provide a low latency, private internet within the internet and allow the user to use their own applications on it, as if nothing ever changed.

Open Source Code

All code to this project can be found on GitHub under the GNU GPL v3 License, allowing you to create your own networks.

Private

To get dedicated servers on Arachnid, only it's MAC address and a unique code tied to your device is shared with the network controllers.

Robust Codebase

The Arachnid Project is a large project, consisting of many programmers around the globe helping to make it useful.

Cross-Compatible

Arachnid Works on virtually any common OS, with APIs for programmers to port applications to exotic hardware.

Interlinked

We peer with several companies, including (but not limited to) Akamai, Cloudflare, etc. And we serve as an Internet Exchange Point.

It's Free!

Anyone can use Arachnid, by simply running an installer. Now, what will you do with it?
